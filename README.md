# Nova
一个简单的编译器
**********************************************
### 介绍

学习完编译原理后突然对这方面产生了兴趣，抱着学习的目的开始用C++实现一个编译器。

### 进展

- 词法分析部分，目前可识别的符号包括：
  - 无符号整型数
  - 浮点数
  - 赋值号
  - 加 减 乘 除 取余
  - 等于号 不等于号 大于号 小于号 大于等于号 小于等于号
  - 与 或 非
- 基于RL(1)的语法分析部分

### 环境

- Visual Studio 2015
